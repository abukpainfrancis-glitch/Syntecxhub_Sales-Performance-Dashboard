## 📌 Project Overview

This project was completed as part of the **Syntecxhub Internship Program**. It involves a comprehensive **Sales Analysis** of a Super Store dataset using **Power BI**, featuring interactive dashboards that reveal key business insights across sales, profit, regions, categories, and shipping modes.

---

## 🎯 Objectives

- Analyze total sales, quantity, profit, and delivery performance
- Compare sales and profit trends Year-over-Year (2019 vs 2020)
- Break down performance by region, segment, category, and ship mode
- Forecast future sales using Power BI's forecasting feature
- Present findings through a clean, interactive dashboard
